# izenak.eus

Aurkitu izena zure seme-alabentzat.

## Project setup

```bash
npm install
```

### Compiles and hot-reloads for development

```bash
npm run serve
```

### Compiles and minifies for production

```bash
npm run build
```

### Lints and fixes files

```bash
npm run lint
```

### Run the unit tests

```bash
npm test
```
